Cada um tem um tempo de travessia:

aluno: leva 1 minuto para cruzar a ponte
professora: leva 2 minutos
zelador: 5 minutos
cientista: 10 minutos

O problema é: qual a sequência de travessia para que o tempo total seja no máximo de 17 minutos.

Contudo, para cruzar a ponte é preciso usar uma lanterna e no máximo duas pessoas podem cruzar a ponte. 
Qualquer pessoa cruzando a ponte precisa segurar a lanterna.

Trabalho avaliado é a modelagem:
	1) Estados
	2) Regras de Transição
	3) Restrições
	4) Visitados
	5) Função objetivo

Estados 1{
    Usar 4 variaveis ints com o valor dos tempos; int aluno = 1; int professora = 2; int zelador = 5; int cientista = 10;
    Variavel para armazenar quem vai ir: int pessoa1 = 0; int pessoa2 = 0;
    Variavel da direcao: char lanterna ['D', 'E'];
    Para controle, tempo que levam, sao duas variaveis, limite e a atual. Mas o tempo maior que deve ser adicionado para o tempo atual. int tempoAtual = 0; int tempoLimite = 17;
    Para controle, ter variavel int para armazenar quantas pessoas estao nas laterais: int quantidadePessoasEsquerda = 4; int quantidadePessoasDireita = (4 - quantidadePessoasEsquerda);
    Estado inicial 
}

Regras de Transição 2{
    Mover no maximo dois em qualquer combinacao;

    Podem ir da direita para a esquerda;
    Levar aluno,
        aluno, professora;
        aluno, zelador;

}

Restrições 3{

    if (tempoAtual > tempoLimite):
