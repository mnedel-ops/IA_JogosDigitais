Cada um tem um tempo de travessia:

aluno: leva 1 minuto para cruzar a ponte
professora: leva 2 minutos
zelador: 5 minutos
cientista: 10 minutos

O problema é: qual a sequência de travessia para que o tempo total seja no máximo de 17 minutos.

Contudo, para cruzar a ponte é preciso usar uma lanterna e no máximo duas pessoas podem cruzar a ponte. 
Qualquer pessoa cruzando a ponte precisa segurar a lanterna.

Trabalho avaliado é a modelagem:
	1) Estados
	2) Regras de Transição
	3) Restrições
	4) Visitados
	5) Função objetivo

    
O que entregar?
	- colocar em um arquivo .md ou .txt
	- disponibilizar no github pessoal (ou outro ambiente de controle de versão) ou enviar para alexz@ufn.edu.br (Assunto: Trabalho1 IA)

Estados 1{
    Usar 4 constantes ints com o valor dos tempos, pois não vão alterar. Os tempos sao: aluno = 1; int professora = 2; int zelador = 5; int cientista = 10.
    Variavel para as posicoes atuais: esquerda = [aluno, professora, zelador, cientista]; direita = [].
    Variavel para a posicao atual da lanterna: lanterna = ['D', 'E'];
    Tempo que levam e o tempo limite (constante). tempoAtual = 0; tempoLimite = 17;
    
    Estado inicial: esquerda = [aluno, professora, zelador, cientista], direita = [], lanterna = E, tempoAtual = 0;
    Estado final: esquerda = [], direita = [aluno, professora, zelador, cientista], lanterna = D, tempoAtual <= 17;
}

Regras de Transição 2{
    Elas podem mover da direita para a esquerda e vice-versa.
    Pode mover ou uma ou duas pessoas, sendo a mais lenta precisa estar com a lanterna;
    Só pode mover se as pessoas estiverem no mesmo lado que a lanterna;
    O tempo gasto é o de quem estiver com a lanterna.
}

Restrições 3{
    Maximo de duas pessoas podem se mover, sendo que a mais lenta carrega a lanterna. Maximo de duas estao se movendo?
    Para se mover, alguem precisa carregar a lanterna, ou estar junto de quem a carregar. Pessoas estao com a lanterna?
    O tempo das total das travessias é menor, ou igual, que 17?
}

Visitados 4{
    Gravar quem esta em cada lado, em que lado a lanterna está e somar ao tempoAtual, o tempo de quem carregou a lanterna.
    Salvar como: (esquerda[], direita[], lanterna, tempoAtual).
}

Função objetivo 5{
    Quando todos sairem do lado esquerdo e irem para o direito, com no maximo 17 minutos.
}
